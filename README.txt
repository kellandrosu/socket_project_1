Chat Server:
    - Use python3 to run server.
    - Requires single argument for port number
    - Usage:
        python3 chatserv.py [port]

Chat Client:
    - compile by calling "make"
    - Makefile will create executible labeled "chatclient"
    - Requires hostname and port number as arguments
    - Usage:
        ./chatclient [hostname] [port]
